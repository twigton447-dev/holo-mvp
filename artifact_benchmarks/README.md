# Artifact Benchmarks

Reusable no-live-first harness for artifact-quality benchmarks. Core claims: `loop_uplift` and `holo_advantage`. Diagnostic POC outputs are not benchmark credit and are not public claims.

Benchmark lanes:

- `holo_factory/`: creates synthetic-but-realistic frozen HoloBuild domain kits.
- `holo_research/`: governs research-agent arms that turn raw material into question-led, source-grounded packets for HoloBuild.
