# v5.1 Changes From v5

Status: candidate note. This does not lock v5.1.

v5.1 tightens v5 in five ways:

1. Active D5 proof comparison is reduced to two anonymous artifacts: ARTIFACT_001 and ARTIFACT_003. ARTIFACT_002 remains diagnostic/ablation only and is excluded from this trial packet.
2. Expert-survivability gates are explicit. Any failed gate creates an 83 ceiling.
3. Risk suppression is elevated to a weighted 15-point category and a required gate.
4. Hard caps now cover false attribution, reliance-critical caveats, stale evidence treated as decisive, draft/provisional source overclaiming, material negative-space misses, missing decision recommendation, and missing stop/go logic.
5. Pairwise outputs must use exact labels only: ARTIFACT_001, ARTIFACT_003, or TIE. Prose pairwise winners are invalid.

v5.1 remains candidate until the D5 blind trial results are reviewed.
