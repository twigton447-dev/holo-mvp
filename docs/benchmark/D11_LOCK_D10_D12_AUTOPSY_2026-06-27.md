## Architecture Lesson

D11-lock Holo is strong at source-boundary, uncertainty, dependency, and adversarial reasoning. It still needs deterministic enforcement for mechanical gates.

The next architecture patch should add a deterministic form actuator between Gov diagnosis and final artifact selection:

1. Local gate computes exact word-band and section defects.
2. Gov routes the next worker with exact section quotas.
3. The final worker receives hard per-section word targets.
4. If the final worker misses the word band, the lane fails closed or enters an explicitly logged deterministic form-normalization step.

This should be treated as a Holo hardening target, not as permission to weaken deterministic gates.
